capytaine.io.bemio module¶

Output of the result as BEMIO hdf5 file.

[1]: https://wec-sim.github.io/WEC-Sim/advanced_features.html#bemio

capytaine.io.bemio.to_bemio_file(dataset, body, filepath)[source]¶

Write a dataset to BEMIO hdf5 file format. For the moment only a single body with 6 rigid body dofs is supported.

Parameters:
  • dataset (xarray.Dataset) – the dataset returned by capytaine.results.assemble_dataset
  • body (FloatingBody) – the studied floating body
  • filepath – path of the hdf5 file
