capytaine.mesh.meshes_collection module¶

Storing a set of meshes.

class capytaine.mesh.meshes_collection.CollectionOfMeshes[source]¶

Bases: tuple, capytaine.tools.geometry.Abstract3DObject

A tuple of meshes. It gives access to all the vertices of all the sub-meshes as if it were a mesh itself. Collections can be nested to store meshes in a tree structure.

Parameters:
  • meshes (Mesh or CollectionOfMeshes)
  • name (str, optional) – a name for the collection
