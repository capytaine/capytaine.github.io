capytaine.io.xarray module¶

capytaine.io.xarray.merge_complex_values(ds: xarray.core.dataset.Dataset) → xarray.core.dataset.Dataset[source]¶

Return a new Dataset where real-valued arrays of shape (2, …) have been replaced by complex-valued arrays of shape (…). Invert of separate_complex_values().

capytaine.io.xarray.separate_complex_values(ds: xarray.core.dataset.Dataset) → xarray.core.dataset.Dataset[source]¶

Return a new Dataset where complex-valued arrays of shape (…) have been replaced by real-valued arrays of shape (2, …). Invert of merge_complex_values().
